# How to Reverse a Linked List Process

*Created by Yohan Hmaiti and Matthew Gibbons.*

## Overview
```
Before:

+------+
| Head | 
+------+                                                                       
    |                                                                          
    v                                                                          
+-------+    +-------+    +-------+    +-------+    +-------+    +------+      
|   1   | -> |   2   | -> |   3   | -> |   4   | -> |   5   | -> | null |      
+-------+    +-------+    +-------+    +-------+    +-------+    +------+      


After:
                                                                        
+------+    +-------+    +-------+    +-------+    +-------+    +-------+      
| null | <- |   1   | <- |   2   | <- |   3   | <- |   4   | <- |   5   |     
+------+    +-------+    +-------+    +-------+    +-------+    +-------+      
                                                                    ^          
                                                                    |          
                                                                 +------+      
                                                                 | Head | 
                                                                 +------+   
```
You woke up one morning to yet another program you wrote in your sleep. 
Assign left=root (A) and right=root.next (B)

```
               Left          Right
            +-------+     +-------+   
            |       | \ / |       |    
       +--> |   A   | -X> |   B   | ---+
       |    |       | / \ |       |    |
       |    +-------+     +-------+    |
       |                               |
       |                               v
   original                        +-------+
     root                          |       |
                                   |   C   | 
                                   |       |
                                   +-------+
                                       |
                                       |
            +-------+     +-------+    | 
            |       |     |       |    | 
            |   E   | <-- |   D   | <--+
            |       |     |       |    
            +-------+     +-------+   
       
```

Assign temp=right->next; (C)

```
               Left          Right
            +-------+     +-------+   
            |       | \ / |       |    
            |   A   | -X> |   B   | ---+
            |       | / \ |       |    |
            +-------+     +-------+    |
                                       |
                                       v
                                   +-------+
                                   |       |
                                   |   C   |  Temp
                                   |       |
                                   +-------+
                                       |
                                       |
            +-------+     +-------+    | 
            |       |     |       |    | 
            |   E   | <-- |   D   | <--+
            |       |     |       |    
            +-------+     +-------+   
                            
```

right->next = left (B)

B now points to A. Remember the red arrows are for the reversed list :)

Move left, right and temp to the following node for each one (to the right by 1)!

```
                            Left
            +-------+     +-------+   
            |       | \ / |       |   \ /
            |   A   | -X> |   B   | ---X
            |       | / \ |       |   /|\
            +-------+     +-------+    |
                ^           |          |
                |           |          v
                |           |      +-------+
                +-----------+      |       |
                                   |   C   |  Right
                                   |       |
                                   +-------+
                                       |
                                       |
            +-------+     +-------+    | 
            |       |     |       |    | 
            |   E   | <-- |   D   | <--+
            |       |     |       |    
            +-------+     +-------+   
                             Temp    
```

right->next = left (C)

C now points to B!

```
                            Left
            +-------+     +-------+   
            |       | \ / |       |   \ /
            |   A   | -X> |   B   | ---X
            |       | / \ |       |   /|\
            +-------+     +-------+    |
                ^           |  ^       |
                |           |  |       v
                |           |  |   +-------+
                +-----------+  +---|       |
                                   |   C   |  Right
                                   |       |
                                   +-------+
                                       |
                                       |
            +-------+     +-------+    | 
            |       |     |       |   \|/
            |   E   | <-- |   D   | <--X
            |       |     |       |   / \
            +-------+     +-------+   
                             Temp    
```

Shift all our nodes by 1 to the right!

```
                          
            +-------+     +-------+   
            |       | \ / |       |   \ /
            |   A   | -X> |   B   | ---X
            |       | / \ |       |   /|\
            +-------+     +-------+    |
                ^           |  ^       |
                |           |  |       v
                |           |  |   +-------+
                +-----------+  +---|       |
                                   |   C   |  Left
                                   |       |
                                   +-------+
                                       |
                                       |
            +-------+     +-------+    | 
            |       |     |       |   \|/
            |   E   | <-- |   D   | <--X
            |       |     |       |   / \
            +-------+     +-------+   
               Temp         Right    
```

right->next = left (D)

D now points to C!

```
                          
            +-------+     +-------+   
            |       | \ / |       |   \ /
            |   A   | -X> |   B   | ---X
            |       | / \ |       |   /|\
            +-------+     +-------+    |
                ^           |  ^       |
                |           |  |       v
                |           |  |   +-------+
                +-----------+  +-- |       |
                                   |   C   |  Left
                               +-> |       |
                               |   +-------+
                               |       |
                               |       |
            +-------+     +-------+    | 
            |       | \ / |       |   \|/
            |   E   | <X- |   D   | <--X
            |       | / \ |       |   / \
            +-------+     +-------+   
               Temp         Right    
```

Shift all the nodes by 1 to the right!

```
                          
            +-------+     +-------+   
            |       | \ / |       |   \ /
            |   A   | -X> |   B   | ---X
            |       | / \ |       |   /|\
            +-------+     +-------+    |
                ^           |  ^       |
                |           |  |       v
                |           |  |   +-------+
                +-----------+  +-- |       |
       Temp = null                 |   C   | 
                               +-> |       |
                               |   +-------+
                               |       |
                               |       |
            +-------+     +-------+    | 
            |       | \ / |       |   \|/
            |   E   | <X- |   D   | <--X
            |       | / \ |       |   / \
            +-------+     +-------+   
              Right          Left    
```

right->next = left, so now E now points to D!

```
                          
            +-------+     +-------+   
            |       | \ / |       |   \ /
            |   A   | -X> |   B   | ---X
            |       | / \ |       |   /|\
            +-------+     +-------+    |
                ^           |  ^       |
                |           |  |       v
                |           |  |   +-------+
                +-----------+  +-- |       |
       Temp = null                 |   C   | 
                +-----------+  +-> |       |
                |           |  |   +-------+
                |           |  |       |
                |           v  |       |
            +-------+     +-------+    | 
            |       | \ / |       |   \|/
            |   E   | <X- |   D   | <--X
            |       | / \ |       |   / \
            +-------+     +-------+   
              Right          Left
```
Then return the root :)!

```
                          
            +-------+     +-------+   
            |       | \ / |       |   \ /
            |   A   | -X> |   B   | ---X
            |       | / \ |       |   /|\
            +-------+     +-------+    |
                ^           |  ^       |
                |           |  |       v
                |           |  |   +-------+
                +-----------+  +-- |       |
    Right = null                   |   C   | 
                +-----------+  +-> |       |
                |           |  |   +-------+
                |           |  |       |
                |           v  |       |
            +-------+     +-------+    | 
            |       | \ / |       |   \|/
    Root -> |   E   | <X- |   D   | <--X
            |       | / \ |       |   / \
            +-------+     +-------+   
               Left        
```
